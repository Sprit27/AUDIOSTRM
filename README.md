# **Instructions for Tools.**<br><br>

<ins>**Clone the repo**</ins>
```shell
   $ git clone https://github.com/Sprit27/AUDIOSTRM.git
```
<br><br>
<ins>**After cloning the repo**</ins><br> 
Install the libraries in the same folder.

```shell
   $ cd /path/to/folder
   $ pip install -r requirements.txt
```
<br><br>

<ins>**Vlc**</ins><br>
You might require the VLC media player in your PC.

Windows--<br>
If you are on Windows, its mostly preinstalled<br>

Linux--<br>
On linux you need to install it sepratly...Open a new terminal(Don't close existing one)
```shell
  $ sudo apt install vlc
```
Then you can close the second terminal.<br><br><br>


<ins>**Stream music**</ins><br>
After instaling the requirements and ensuring you have the vlc player, you can stream music in background and ads free.<br>
Start the tool by--
```shell
   $ python yt3.py
```
it will ask for playlist or single video url, past it and press enter and you are good to go....<br><br>

<ins>**Download track**</ins><br>
After installing the requirements you can download the track.<br>
Start the tool by--
```shell
  $ python yt2.py
```
It will ask for playlist or vedio url. Onece you paste it it will download all the tracks in the playlist or the single track in the same folder.
